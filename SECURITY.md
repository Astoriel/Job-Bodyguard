# Security And Privacy Notes

**Snapshot date:** 2026-02-20

- Job descriptions, resume text, and application notes can be sensitive personal data.
- Review settings before enabling any OpenAI-compatible endpoint or remote dashboard sync.
- The local extension path should prefer Chrome Storage/local processing where possible.
- Do not publish builds with personal API keys, Supabase credentials, or resume samples.
- The repository is in maintenance mode and should be security-reviewed before redistribution.
