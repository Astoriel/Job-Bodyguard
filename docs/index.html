Install from source

Current distribution model

  1. Clone the repository and run `pnpm install`.
  2. Build the extension with `pnpm --filter @job-bodyguard/extension build`.
  3. Open `chrome://extensions`, enable developer mode, and load `apps/extension/dist`.
  4. Optionally download a packaged zip from the GitHub releases page.
