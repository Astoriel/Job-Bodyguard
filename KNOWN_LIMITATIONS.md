# Known Limitations

**Snapshot date:** 2026-02-20

- The extension is in maintenance mode and may need updates for changing job-board DOM structures.
- Parser coverage is intentionally limited to the implemented sources.
- The web dashboard and Supabase/OpenAI paths should be reviewed before production use.
- Privacy-sensitive resume and job content flows need explicit user configuration and disclosure.
- Automated tests are currently limited; parser and extension smoke coverage should be expanded before renewed distribution.
