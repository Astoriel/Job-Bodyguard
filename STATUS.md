# Project Status

**Status:** Maintenance mode  
**Snapshot date:** 2026-02-20  
**Primary audience:** developers who want to inspect or fork a job-search browser extension.

`Job-Bodyguard` is a Chrome extension and web dashboard prototype for parsing job pages, flagging risk signals, saving applications, and optionally using AI-assisted analysis. The extension code and UI are the core artifact; the project is not presented as an actively maintained commercial product.

## Current Confidence

- Working: browser extension structure, parsers for common job boards, popup/sidepanel/dashboard UI, local saved jobs flow.
- Planned: stronger parser tests, documented privacy model, refreshed extension packaging, maintained web dashboard deployment.
- Not claimed: active Chrome Web Store availability, guaranteed compatibility with all job boards, production SaaS backend.
